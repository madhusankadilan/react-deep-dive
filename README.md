# poc-react
POC repo for React JS concepts and features

## React (Fiber, Hooks, & Architecture)

### Beginner
Flash Cards App: Basic state management and conditional rendering.

Compound Component Tabs: Using Context for implicit state sharing.

Custom Hook Library: Building useLocalStorage, useWindowSize, etc.

Recursive Tree View: Rendering nested folder structures.

Theme Switcher: Global state management with useContext.

Animated List Transitions: Using framer-motion with React state.

Pomodoro Timer: Handling side effects with useEffect.

Quiz App: Managing a complex state machine for questions/answers.

GitHub Random Repo Finder: State-based API fetching.

Book Tracking Library: CRUD operations with an external API.

Theme implementation. (Light theme, Dark theme)

### Intermediate
Optimistic UI Comment System: Instant UI updates with "rollback" logic.

Virtualized List: Rendering only visible rows for performance.

State Management Middleware: Building a logger for useReducer.

Declarative Modal Portal: Using createPortal for accessible overlays.

Dynamic Form Builder: Generating UI from a JSON schema.

Kanban Board: Drag-and-drop state updates and persistence.

E-commerce Product Gallery: Complex filtering and sorting logic.

Blog with Contentful CMS: Integrating React with a Headless CMS.

Multi-Step Survey: State persistence across multiple "pages."

Poll/Voting App: Real-time UI updates via WebSockets.

### Advanced
Custom Virtual DOM & Reconciler: Implementing render(element, container).

Fiber Work-Loop Simulation: A recursive loop for processing the Fiber tree.

Concurrent Data Fetching: Implementing "Render-as-You-Fetch" with Suspense.

Custom SSR Engine: Building a manual server-side renderer.

State Management via Proxy: Building a Zustand/Valtio clone.

Headless UI Library: Building accessible, unstyled components.

React Profiler Dashboard: Tracking commit times and re-renders.

Atomic State Library: Building a Recoil/Jotai clone.

Canvas Drawing Engine: Syncing React state with the Canvas API.

Offline-First App: Using TanStack Query with persistent storage.

Video Editor Timeline: High-performance UI for frame scrubbing.

Streaming GraphQL Client: Handling real-time chunked data.

Feature Flag Provider: Context-based staged rollouts.

Custom CSS-in-JS Engine: Parsing template literals into style tags.

Real-time Collaborative Board: Integrating React with CRDTs (Yjs).

AI Chat Interface: Handling streaming LLM responses in the UI.

High-Performance Stock Ticker: Optimizing 1000+ updates/sec.

Server Components Prototype: Simulating React Server Components (RSC).

React-Based Browser Emulator: UI that mimics a browser window.

V8 Lifecycle Visualization: An interactive React app showing V8's memory stages.

Micro-Frontend Orchestrator: Loading isolated apps into one shell.

Custom Hydration Engine: Manually attaching React events to static HTML.

Role-Based Access Control (RBAC) Wrapper: High-order components for auth.

Global Command Palette: (Cmd+K) search and action interface.

Production-Grade Book Management: A full microservices simulation for ECS/EKS.
